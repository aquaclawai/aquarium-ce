<objective>
Extend template import (instantiateTemplate) to re-evaluate trust policy for each extension declaration synchronously at instantiation time, insert lifecycle rows only for extensions that pass, omit blocked extensions with warnings, and update setup command generation to skip lifecycle-managed extensions.

Purpose: When a template is imported, the trust landscape may have changed since export. Extensions that were safe at export time may now be compromised, revoked, or unverified. Trust re-evaluation at import time ensures the target instance only gets extensions that pass current security policy. evaluateTrustPolicy works from DB trust overrides and source type alone — no running gateway required.

Output: Modified instantiateTemplate that processes TemplateExtensionDeclaration[], evaluates trust synchronously, inserts lifecycle rows only for passing extensions, and returns blocked/override-required extension warnings.
</objective>

4. **CRITICAL — Synchronous trust evaluation per locked CONTEXT.md decisions.** For each extension in `templateExtensions`, call `evaluateTrustPolicy` synchronously (it is async but does NOT need a running gateway — it reads DB trust overrides and computes tier from source type):

   ```typescript
   for (const ext of templateExtensions) {
     // evaluateTrustPolicy with null signals:
     //   bundled → allow (tier=bundled)
     //   non-bundled + null signals → tier=unscanned → block
     //   community with existing DB override → allow
     const evaluation = await evaluateTrustPolicy(
       instance.id,
       ext.id,
       ext.kind,
       ext.source,
       null  // No gateway running → no ClawHub signals available
     );

     if (evaluation.decision === 'block') {
       if (evaluation.tier === 'community') {
         // Community extension without admin override on THIS instance
         requiresTrustOverride.push({
           id: ext.id,
           kind: ext.kind,
           source: ext.source,
           reason: evaluation.blockReason ?? 'Community extension requires admin trust override',
         });
       } else {
         // unscanned / scan-failed → hard block per CONTEXT.md: "cannot instantiate"
         blockedExtensions.push({
           id: ext.id,
           kind: ext.kind,
           reason: evaluation.blockReason ?? 'Extension blocked by trust policy',
         });
       }
       continue; // Do NOT insert lifecycle row for blocked extensions
     }

     // evaluation.decision === 'allow' → insert lifecycle row
     // ... (see step 5)
   }
   ```

   - Community without override: returned in requiresTrustOverride[], NOT inserted
   - Community WITH existing override: allowed, inserted
