**TrustBadges.tsx** — New component rendering inline trust signal badges on catalog rows.

```typescript
interface TrustBadgeRowProps {
  trustTier?: TrustTier;
  trustSignals?: TrustSignals;
  source: 'bundled' | 'clawhub';
}
```

Export `TrustBadgeRow` function component:
- If source === 'bundled': render single badge `<span class="trust-badge trust-badge--bundled">Bundled</span>` (using i18n key extensions.trust.bundled) and return early
- If trustTier === 'verified': render green checkmark badge `<span class="trust-badge trust-badge--verified">{checkmark} {t('extensions.trust.verified')}</span>`
- If trustTier === 'community': render yellow badge `<span class="trust-badge trust-badge--community">{t('extensions.trust.unverified')}</span>`
- If trustTier === 'unscanned': render red badge `<span class="trust-badge trust-badge--unscanned">{t('extensions.trust.scanFailed')}</span>`
- Then render signal badges if trustSignals exists:
  - If virusTotalPassed === true: green shield `<span class="trust-badge trust-badge--scanned">{shield} {t('extensions.trust.scanned')}</span>`
  - If virusTotalPassed === false: red shield `<span class="trust-badge trust-badge--scan-failed">{shield} {t('extensions.trust.scanFailed')}</span>`
  - If downloadCount > 100: `<span class="trust-badge trust-badge--downloads">{downloadCount} downloads</span>`
  - If ageInDays > 90: `<span class="trust-badge trust-badge--age">{ageInDays}d</span>`

Use HTML entities or simple text characters for icons (checkmark: &#10003;, shield: &#128737;). Do NOT use external icon libraries.
