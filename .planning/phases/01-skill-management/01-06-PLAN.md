**CredentialConfigPanel.tsx** — Inline panel for configuring extension credentials.

Props:
```typescript
interface CredentialConfigPanelProps {
  instanceId: string;
  skillId: string;
  skillName: string;
  status: ExtensionStatus;
  onClose: () => void;
  onSaved: () => void;
  disabled: boolean;
}
```

State: `field: string` (input), `value: string` (input), `saving: boolean`, `error: string | null`, `success: boolean`

Render:
- A card/panel that appears below or beside the skill row (inline expansion, not a modal)
- Title: "Configure {skillName}" 
- If status === 'installed': show message "This skill requires credentials to activate."
- Form fields:
  - Credential field name: text input (e.g., "OPENAI_API_KEY") — pre-fill from skill config if available
  - Value: password input (type="password") for the API key
- Save button: POST to `/instances/${instanceId}/extension-credentials` with body:
  ```typescript
  {
    provider: skillId,
    credentialType: 'api_key',
    value: value,
    extensionKind: 'skill',
    extensionId: skillId,
    targetField: field
  }
  ```
- On success: show checkmark, call `onSaved()` to refresh parent
- On error: show error message inline
- Close button (X) to collapse panel

Use `useTranslation()` for strings. Add to locale files:
```json
"credentials": {

  "requiresCredentials": "This skill requires credentials to activate.",
  "fieldLabel": "Credential Field",
  "valueLabel": "API Key / Value",
  "save": "Save Credential",
  "saving": "Saving...",
  "saved": "Credential saved successfully",
  "saveFailed": "Failed to save credential"
}
```
Add these strings to all 6 locale files with proper translations.
