### A — Translate per locale

For each of zh.json, fr.json, de.json, es.json, it.json: find the `issues.detail.*` + `chat.*` subtree (installed in Wave 0 with en placeholders) and replace each string value with a native translation.

**Terminology guidance (for consistency with Phase 23):**

| English | zh | fr | de | es | it |
|---------|-----|-----|-----|-----|-----|
| Issue | 议题 | Ticket | Vorgang | Issue | Issue |
| Comment | 评论 | Commentaire | Kommentar | Comentario | Commento |
| Task | 任务 | Tâche | Aufgabe | Tarea | Attività |
| Agent | 代理 | Agent | Agent | Agente | Agente |
| Cancel | 取消 | Annuler | Abbrechen | Cancelar | Annulla |
| Delete | 删除 | Supprimer | Löschen | Eliminar | Elimina |
| Running | 运行中 | En cours | Läuft | En ejecución | In esecuzione |
| Completed | 已完成 | Terminé | Abgeschlossen | Completado | Completato |
| Failed | 失败 | Échec | Fehlgeschlagen | Fallido | Fallito |

**Rules:**
- Preserve ALL `{{placeholder}}` interpolation tokens verbatim — `{{count}}`, `{{relativeTime}}`, `{{number}}`, etc. MUST NOT be translated.
- Preserve the ellipsis character `⋯` (U+22EF) in the truncated marker — same Unicode, all locales.
- Keyboard shortcut hints (`⌘⏎ to send`) should stay English in zh/ja but can localize in fr/de/es/it if the convention there allows (keep English if uncertain).
- Do NOT change the key names. Do NOT add new keys. Do NOT remove keys. Parity script enforces structure.
- Character count examples: `issues.detail.task.truncated` en = `"⋯ truncated (showing {{shown}} of {{total}} bytes)"` → zh = `"⋯ 已截断（显示 {{shown}} / {{total}} 字节）"`.

**Strategy:**

Because all 5 locales need the same ~70-key subtree translated and the parity script is the objective gate, the implementer can:
1. Read en.json to extract the `issues.detail.*` + `chat.*` subtree.
2. For each locale, find the same subtree + overwrite each string value. Use a text editor; mass find-replace is NOT safe (JSON escape mismatches).
3. After each locale, run `node apps/web/scripts/check-i18n-parity.mjs` to confirm parity holds.
